# AgriConnect 🌾

## Smart Agricultural Land Rental Platform

AgriConnect is a web application that connects landowners and farmers for agricultural land leasing.

## Features

* User Registration
* User Login
* Add Land
* View Available Lands
* Rent Agricultural Land
* MongoDB Database Integration

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* MongoDB
