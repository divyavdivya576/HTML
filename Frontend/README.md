# AgriConnect 🌾

## Smart Agricultural Land Rental Platform

AgriConnect is a full-stack web application that connects landowners and farmers through a digital platform. Landowners can add agricultural lands for rent, and farmers can browse available lands and submit rental requests.

---

## Features

✅ User Registration

✅ User Login Authentication

✅ Add New Agricultural Land

✅ View Available Lands

✅ Rent Land Request Form

✅ MongoDB Database Integration

✅ Responsive Dashboard

---

## Project Screenshots

### Home Page

This page introduces AgriConnect and allows users to navigate to Login and Registration pages.

### Login Page

Users can securely log in using their registered credentials.

### Register Page

New users can create an account by providing personal details.

### Dashboard Page

Displays all available agricultural lands with details such as owner name, location, area, rent cost, soil type, water availability, and contact information.

### Add Land Page

Landowners can add new agricultural land details to the system.

### Rent Land Page

Farmers can submit rental requests for available lands.
